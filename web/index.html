This project provides Magic the Gathering card images and set and mana symbols for developers to easily use in their projects.

For more details see: Documentation and Discussion Thread

Documentation(back to top)

Card Images

Most images are in 480x680 resolution and come from CCGHQ.

Lower resolution images are provided until a higher resolution is available.

Reference the set list table to see which resolution each set is currently in.

Some cards may be in a different resolution, such as the 'plane' cards in the Planechase sets.

URLs are case insensitive and multiple naming formats are supported:

  • Card Names   Supported
  • MTGJSON      Supported
  • Cockatrice   Supported
  • CCGHQ        Supported*

* A few sets are not available, see: Missing Sets


Images are available via 3 different methods:

Access By Set


Format: http://mtgimage.com/set/<set code>/<card name>.jpg

or

Format: http://mtgimage.com/setname/<set name>/<card name>.jpg


Examples:
  • Sen Triplets  http://mtgimage.com/set/ARB/Sen Triplets.jpg
  • Archweaver    http://mtgimage.com/set/RTR/Archweaver.full.jpg
  • Harrow        http://mtgimage.com/setname/Zendikar/Harrow.jpg

You can browse the set images via set code or set name

Even though when browsing names are lowercase, you can access them with different cases/formats.



Access By Card


Format: http://mtgimage.com/card/<card name>.jpg

Examples:
  • Craw Wurm    http://mtgimage.com/card/Craw Wurm.jpg
  • Serra Angel  http://mtgimage.com/card/Serra_Angel.jpg

The card used comes from the latest set it was printed in, unless an earlier set has a higher resolution.

Token images are not available this way. You need to access those by set/setname.



Access By multiverseid


Format: http://mtgimage.com/multiverseid/<multiverseid>.jpg

Examples:
  • Elspeth, Knight-Errant   http://mtgimage.com/multiverseid/174859.jpg
  • Akroma's Memorial        http://mtgimage.com/multiverseid/279712.jpg



Feel free to directly link (hotlink) to the images with your website or app.

The images are hosted on a dedicated server with a gigabit connection and tons of bandwidth.


Crops

For artwork crops instead of full cards, change the image name to end with '-crop.jpg' or '.crop.jpg' instead of '.jpg'

Like cards, crops are slightly compressed. For uncompressed crops the image name should end with '.crop.hq.jpg'

Crops have various resolutions.


Split Cards

Split cards like 'Fire // Ice' can be accessed in multiple ways:
  • fireice.jpg
  • fire ice.jpg
  • fire.jpg
  • ice.jpg

Image Compression

The card images have been compressed a bit to save on bandwidth.

If you want to access the full quality, uncompressed image, change the image name to end with '.hq.jpg' instead of '.jpg'

Note: There is very little visual difference between the two.

Example Chandra's Phoenix.jpg (95k) vs Chandra's Phoenix.hq.jpg (471k)

Mana and Set Symbols(back to top)

Symbols are available in several sizes and formats.

Formats: PNG, GIF, SVG

Sizes: 8, 16, 24, 32, 48, 64, 96, 128, 256, 512, 768, 1024

Each size is the maximum width or height of the symbol image (eg 32x27 or 127x128).

URLs are case insensitive.



Set Symbols


Format: http://mtgimage.com/symbol/set/<set code>/<rarity>/<size>.<format>

Examples:

  • Torment Rare    64 GIF   http://mtgimage.com/symbol/set/tor/r/64.gif
  • Theros  Mythic 128 PNG   http://mtgimage.com/symbol/set/ths/m/128.png
  • Odyssey Common 256 PNG   http://mtgimage.com/symbol/set/ody/c/256.png



 SVG Format: http://mtgimage.com/symbol/set/<set code>/<rarity>.svg
SVG Example: Ice Age Uncommon SVG   http://mtgimage.com/symbol/set/ice/u.svg


<rarity> is a single letter representing the rarity: c, u, r, m, s

You can also use the full rarity: common, uncommon, rare, mythic, special

Use rarity 'c' for basic lands.

Use rarity 's' for Time Spiral "Timeshifted" cards (TSB) and Vanguard (VAN).

The set list denotes which sets have which rarities.

Set sybmols are rendered in 'accurate' style.



Mana Symbols


Format: http://mtgimage.com/symbol/mana/<mana code>/<size>.<format>

Examples:

  • Green 64 GIF           http://mtgimage.com/symbol/mana/g/64.gif
  • 10 Colorless 128 PNG   http://mtgimage.com/symbol/mana/10/128.png
  • Blue/White 256 PNG     http://mtgimage.com/symbol/mana/uw/256.png



 SVG Format: http://mtgimage.com/symbol/mana/<mana code>.svg
SVG Example: 2 Colorless/Red SVG   http://mtgimage.com/symbol/mana/2r.svg

Examples:

  • Tap 64 GIF      http://mtgimage.com/symbol/other/t/64.gif
  • Untap 128 PNG   http://mtgimage.com/symbol/other/q/128.png



 SVG Format: http://mtgimage.com/symbol/other/<code>.svg

The card type symbols (enchantment, instant, etc) come from Future Sight.
The power and toughness symbols come from Portal.
The chaosdice and planeswalk symbols come from Planechase.
